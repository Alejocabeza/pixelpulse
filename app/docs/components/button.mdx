## Importación

PixelUI exporta solamente un modelo de Button:

```jsx
import { Button } from "pixelui-button";
```

## Uso
<br/>
<Button color='primary'>Button</Button>
<br/>
<br/>
```jsx
import React from 'react'
import {Button} from 'pixelui-react'

export default function App() {
  return (
    <>
      <Button color="primary">Button</Button>
    </>
  )
}
```
