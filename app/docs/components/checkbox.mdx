## Importación

PixelUI exporta solamente un modelo de Checkbox:

```jsx
import { Checkbox } from "pixelui-react";
```

## Uso
<br/>
<div className='flex flex-row gap-[1rem]'>
  <Checkbox title='option' />
</div>
<br/>

```jsx
import react from 'react'
import { Checkbox } from "pixelui-react";

export const App = () => {
  return (
    <div className='flex flex-row gap-[1rem]'>
      <Checkbox title='option' />
    </div>
  )
}
```
